# eu4ach

```
USAGE:
     eu4ach random:                 random achievement
     eu4ach veasy:                  random very easy achievement
     eu4ach easy:                   random  easy achievement
     eu4ach medium:                 random  medium achievement
     eu4ach hard:                   random hard achievement
     eu4ach very_hard:              random very hard achievement
     eu4ach complete <id : number>: complete achievement
     eu4ach track <id : number>:    tracks the given achievement
     eu4ach current:                current achievement
     eu4ach clear:                  clears the current achievement
```
